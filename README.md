### 人工智能（周正阳）
老师水平还是很高的，就是课程排的感觉不太合理，内容有点多，从挑西瓜一直讲到了vision transformer  
虽然但是感觉给分比较随缘，只给了80分，看不到小分也不知道哪块出了问题
#### 考勤
不定时点名，但是频率不高，感觉也是为了学院给的kpi
#### 平时作业
只有一次（手算adaboost，只列公式不算结果），课上布置带回去做，下次上课交一张纸  
组队完成一个ML或DL有关的项目，最后做PPT上台汇报，git拉一个项目下来或者去Kaggle找一个房价预测之类的做的深入一点都行，本科有相关项目的也可以直接用
#### 实验
四次实验，会给基本代码，只需稍微改动甚至不用改：
  - 线性回归
  - SVM调不同的惩罚因子C、ID3调决策树的高度
  - 配置pytorch
  - U-Net进行语义分割
#### 考核
开卷考试  
题目出的比较有深度，如果没什么ML和DL基础光看PPT复习的话还是会有一点折磨的  
正经写的话真的要写快两个小时  
好在总分只占50%
##### 考试题
不全，只记得这些
1. **选择题**
    1. 四个选项分别列出了一个ML算法及其损失函数，选哪个损失函数对应错误
    2. 中间还有一些概念题？记不清了 
    3. 反卷积的计算
    4. Q-Learning 的简单计算
2. **简答题**
    1. 给出了训练过程中的误差、方差还有一个什么指标的图像，问欠拟合、过拟合分别在图像哪个阶段，要在哪个时刻早停
    2. 介绍一下Google的Inception结构并说明有哪些好处？忘了怎么设问的，大概意思差不多这样
    3. CNN和GCN网络过深分别会出现什么问题，有什么解决办法（习题课讲过）
3. **计算题**
    1. 决策树有关的计算，信息增益之类的，套公式就行
    2. 集成学习选择分类器标准是什么，以及给了一个表格其中有多个分类器的分类结果，计算每个分类器的准确率 $w_i$，在此基础上设计一种方式计算每个分类器的权重
4. **算法题**  
说是算法题其实写思路就行
    1. word2vec
    2. 有一组人脸数据，其中部分label缺失，根据半监督学习的理论设计一个分类算法，并说明**使用了什么损失函数**

#### 平时作业

#### 实验

#### 考核

考的还是偏难的 被橄榄了  
再加上同一天白天出了人工智能成绩 破大防

##### 考试题
1. **选择题**  
2分*10，都比较简单，难度小于等于408数据结构
2. **简答题**  
10分*5
   1. $T(n) = max_{(0 \leq q<n)}(T(q)+T(n-q-1))+ \theta (n)$，证明 $T(n) = (n^2)$ （ $q$ 的范围可能是这样，但 $q$ 的范围是这样不太可能）
   2. 两数之和 要求 $O(nlgn)$
   3. 最大递增子序列 要求 $O(nlgn)$
   4. 指示器变量求帽子问题的期望
   5. 修改迪杰斯特拉算法，使其能够处理负权边
3. **算法设计与分析**  
15分*2
   1. 逆序对问题
      1. 给一个数组写出其所有逆序对
      2. 什么情况下逆序对最多，有多少个
      3. $n$ 个元素随机排列，求逆序对个数的期望
      4. 设计算法求逆序对 要求 $O(nlgn)$
   2. 贪心问题
      1. 找零 $n$ 元，有15、10、5、1四种硬币，设计贪心算法并证明其正确性
      2. 有 $c^0、c^1、...、c^k$ 的硬币，设计贪心算法并证明其正确性
      3. 写出一组硬币，使得贪心算法不能得到最优解
      4. 找零 $n$ 元，硬币有 $k$ 种，设计算法解决 要求 $O(nk)$
